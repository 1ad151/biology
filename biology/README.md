# 生物多样性保护网站

这是一个关于生物多样性保护的教育网站，旨在提高公众对生物多样性重要性的认识，并鼓励人们参与到保护行动中来。

## 项目简介

本网站是一个完整的静态网站项目，包含以下主要内容：

- **首页**：介绍生物多样性的重要性和面临的威胁
- **关于页面**：详细解释什么是生物多样性及其重要性
- **濒危物种页面**：展示多种濒危物种的信息和保护现状
- **生态系统页面**：介绍地球上不同类型的生态系统
- **保护行动页面**：提供个人、社区和机构可以采取的保护措施

## 技术特点

- 纯HTML、CSS和JavaScript实现，无需后端服务器
- 响应式设计，支持各种设备访问
- 现代化的UI设计，美观易用
- 包含动画效果和交互功能
- 内容丰富，具有教育意义

## 使用方法

1. 直接在浏览器中打开 `index.html` 文件即可访问网站
2. 或者使用本地服务器运行（推荐）：
   - 使用Python: `python -m http.server 8000`
   - 使用Node.js: `npx http-server`
   - 然后访问 `http://localhost:8000`

## 网站功能

### 主要功能
- 响应式导航栏，支持移动端菜单
- 数字统计动画效果
- 页面滚动动画
- 平滑滚动效果
- 卡片悬停效果

### AI功能（新增）
- **AI问答助手**：智能回答生物多样性相关问题
- **智能物种搜索**：使用AI技术快速搜索物种信息
- **AI物种识别**：上传图片识别动植物（演示功能）
- **个性化保护建议**：根据用户兴趣推荐保护行动

### 页面内容
- **首页**：展示生物多样性的重要性、统计数据、威胁因素等
- **关于页面**：详细介绍生物多样性的定义、重要性和现状
- **濒危物种页面**：展示9种濒危物种的详细信息
- **生态系统页面**：介绍9种不同类型的生态系统
- **保护行动页面**：提供具体的保护建议和行动指南
- **AI助手页面**：提供AI问答、智能搜索、物种识别和个性化建议功能

## 设计理念

本网站采用绿色主题，象征着自然和生命。设计简洁现代，注重用户体验和教育效果。通过丰富的视觉元素和清晰的内容结构，帮助访问者更好地理解生物多样性的重要性。

## 教育意义

这个网站不仅是一个作业项目，更具有实际的教育意义：

1. **提高意识**：帮助人们了解生物多样性的重要性和面临的威胁
2. **传播知识**：提供关于濒危物种和生态系统的科学知识
3. **鼓励行动**：提供具体的保护措施，鼓励人们参与保护行动
4. **社会责任**：强调每个人在保护生物多样性中的责任和作用

## AI技术说明

本网站集成了多种AI技术演示功能：

1. **自然语言处理 (NLP)**：理解用户提问，提供智能回答
2. **智能搜索算法**：语义搜索，理解搜索意图
3. **图像识别技术**：识别动植物图片（演示功能）
4. **推荐系统**：基于用户偏好推荐保护行动

**注意**：当前AI功能为演示版本，使用规则引擎和模拟数据实现。实际应用中可以接入真实的AI API（如OpenAI、百度AI、腾讯AI等）来提供更强大的功能。
